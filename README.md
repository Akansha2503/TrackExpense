Expense Tracker (MERN Stack)

A full-stack Expense Tracker & Finance Dashboard built using the MERN stack to manage income, track expenses, and visualize financial data.

🚀 Live Demo

🔗 https://trackexpense-frontend.onrender.com

✨ Features

Add, edit, delete transactions
Category-based tracking
Interactive dashboard with charts
Weekly & monthly reports
Secure REST APIs

🛠️ Tech Stack

Frontend: React, Tailwind CSS
Backend: Node.js, Express.js
Database: MongoDB
